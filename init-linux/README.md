## Usage
install git
```
apt-get update
apt-get install -y git
```
clone & run
```
git clone https://github.com/elliotxx/scripts.git
cd scripts/init-linux
sh init-linux.sh
```
然后登录普通账户运行
```
su yym
cd ~
git clone https://github.com/elliotxx/scripts.git
cd scripts/init-linux
sh user-init-linux.sh
```
